Rivet Supply

Pressurises a rivet from ARF-700 to the feed head unit.

Note: Turn on the vacuum switch. Move to the supply point before this node.

I/O Port Configuration

  • DO[0]: Rivet supply request output port
  • DO[1]: Finger open/close operation output port (HIGH: finger close)
  • DO[7]: Error output port
  • DI[0]: Rivet pass sensor input port

Adjustable Parameters

There are 3 sliders for adjusting the following timing parameters:

  • Time to check rivet supply
  • Time to open fingers
  • Time until air stops
